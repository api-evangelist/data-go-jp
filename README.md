# DATA GO JP (data-go-jp)

DATA GO JP is a national government open-data portal for Japan running **CKAN**, exposing a standardized catalog API over ~18,366 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-go-jp/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-go-jp-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.5

## API
- **DATA GO JP CKAN Action API** — base `https://www.data.go.jp/data/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://www.data.go.jp/data)
- Live endpoint verified: `https://www.data.go.jp/data/api/3/action/status_show` · ~18,366 datasets
